# video_splash_screen
A video splash screen for flutter project

This code snippet depends on video_player and chewie, so don't forget to add:

dependencies:
  flutter:
    sdk: flutter
  video_player: ^2.2.5
  chewie: ^1.2.2
  
**and **

flutter:
  assets:
      - assets/videos/intro.mp4
      - assets/images/logo.png
